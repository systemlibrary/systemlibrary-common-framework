# Json

Framework comes with many registered converters out of the box, to simplify the datetime and date parsing of strings to datetime objects, to convert strings and int's to Enums, and with a few empty converters to bypass types that cannot be serialized/deserialized

# Usage
Through string extension: Json() and Json<T>()s

# Note
The converters are registered in both the default serializer options and in the system component model serialization from body and url param data into C# models
